Welcome to Dominance-Analysis documentation!
=============================================

Dominance-Analysis is a Python package built for accurately determining the relative importance of interacting predictors in a regression or a classification model.The variable's individual effect as well as its effect in the presence of other variables is accounted for in identifying its proportionate contribution to the model.

The package is currently compatible with Python 2.7, 3.5 and 3.6.

You can find out more regarding the package in the `Official Dominance Analysis Documentation`_.

.. _Official Dominance Analysis Documentation: https://bhagatsajan0073.github.io/dominance-analysis/

Installation
=============================================

Use the following command to install the package:


```  
pip install dominance-analysis
```
